Домашнее задание № 4

Типы селекторов:

  • по тегу - например: p {}. Будут выбраны элементы с тегом р;
  • по id - #id {}. Будут выбраны элементы с определённым id;
  • по классу - .class {}. Будут выбраны элементы с классом .class;
  • по нескольким классам - .class1.class2 {}. Будут выбраны элементы, в которых указаны классы "class1 class 2";
  • по вложенности - p span {} Будут выбраны элементы, которые находятся в теге span внутри тега p;
  • универсальные - * {}. Будут выбраны все элементы разметки html страницы;
  • дочерние - p > span {}. Будут выбраны элементы тега span, который находится сразу после тега p;
  • соседние - p + span {}. Будут выбраны элементы тега span, только если он является соседним для элемента p и следует сразу после него;
  • родственные - p ~ span {}. По своему поведению похожи на соседние селекторы, но в отличие от них стилевые правила применяются ко всем близлежащим элементам тега span;
  • по атрибуту - p[атрибут] {}. Стиль применяется к тем элементам, внутри которых добавлен указанный атрибут;
  • и т.д.
