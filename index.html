  1. git --version для проверки версии гита
  2. git init для инициализация гит проекта
  3. git status для проверки статуса проекта и просмотра отслеживаемых/не отслеживаемых файлов
  4. git add . для добавления всех файлов
  5. git commit -m "[описание]" для сохранения файлов под коротким описанием
  6. git log просмотр истории своих коммитов
  7. git clone https://github.com/Stanysmaev/fe_basic_homework.git для копирования репозитория с гитхаба на свой компьютер
  8. git remote -v для просмотра местонахождения репозитория
  9. git push origin main для отправки
